# واجهة موقع إشعارك - نسخة محسنة

نسخة من تحسين الواجهة الأمامية لموقع إشعارك، مقدمة بواسطة Preview 249.

## عن التصميم
تم بناء هذه الواجهة بالاعتماد على أسلوب التصميم النظيف والعصري (Clean Modern Design)، مع التركيز على تجربة مستخدم مريحة (UI/UX) ومساحات بصرية واسعة.

## ملاحظات لمطوري الـ Back-end
- جميع الملفات تعمل بنظام (Server-Side Rendering) وهي جاهزة للحقن بأكواد بايثون.
- تم استخدام Classes الخاصة بـ Tailwind بشكل مباشر داخل عناصر HTML.
- يمكنكم العثور على الصفحات الرئيسية مثل (login.html) في المجلد الرئيسي.
